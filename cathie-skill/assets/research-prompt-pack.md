# Cathie.skill Prompt Pack

Use these prompts when you want to start quickly.

## Innovation theme scan

```text
用 cathie-skill 深度分析 [AI/新能源/机器人/基因组学/自动驾驶] 赛道。
判断这个技术处于 S 曲线哪个阶段，成本曲线还需要走多远，
5 年 TAM 的基准情景是多少，找出 5 个最值得优先研究的标的，
说明每家的 S 曲线位置、成本曲线优势、5年情景价值和主要降级条件。
```

```text
Use cathie-skill to deeply analyze the [AI / EV / robotics / genomics / autonomous vehicles] sector.
Determine the S-curve phase, cost curve progress, and 5-year TAM in the base case.
Find the 5 best research priorities, and for each explain:
S-curve position, cost curve advantage, 5-year value scenario, and main downgrade condition.
```

## Single company — full innovation analysis

```text
用 cathie-skill 深度分析 [公司/股票代码]。
联网查渗透率数据、成本基准和专利，判断 S 曲线位置和 Wright's Law 学习率，
估算 5 年 TAM 情景和公司市场份额，给出当前估值隐含的增长率是否合理。
```

```text
Use cathie-skill to deeply analyze [company / ticker].
Check penetration rate data, independent cost benchmarks, and patents.
Assess S-curve position and Wright's Law learning rate.
Estimate the 5-year TAM scenario and addressable market share.
State whether the current valuation implies a reasonable or aggressive growth rate.
```

## Cost curve check

```text
用 cathie-skill 分析 [电池/太阳能/DNA测序/激光雷达] 的成本曲线。
用 Wright's Law 估算什么时候越过下一个关键成本阈值，
这个阈值会解锁什么新用例，以及有哪些材料或制造约束可能打断曲线。
```

```text
Use cathie-skill to analyze the cost curve for [batteries / solar / DNA sequencing / LiDAR].
Apply Wright's Law to estimate when the next critical cost threshold is crossed.
Explain what new use case that unlocks.
Identify any materials or manufacturing constraints that could break the cost curve.
```

## S-curve positioning only

```text
用 cathie-skill 判断 [技术] 在全球和中国的 S 曲线位置。
给出当前渗透率数据、是否已跨越鸿沟的判断、下一个阶段的触发条件，
并说明主流市场采用和早期用户采用有什么本质区别。
```

```text
Use cathie-skill to determine the S-curve position of [technology] globally and in China.
State the current penetration rate, whether the chasm has been crossed, and the next phase trigger.
Explain the difference between early adopter and mainstream adoption for this technology.
```

## Challenge a popular innovation stock

```text
用 cathie-skill 挑战 [公司]。
S 曲线位置真的支持现在的估值吗？成本曲线学习率有独立数据支撑吗？
当前股价隐含了什么增长率？什么情况说明采用速度会大幅低于预期？
```

```text
Use cathie-skill to challenge [company].
Does the S-curve position actually support the current valuation?
Is the cost curve learning rate backed by independent data?
What growth rate does the current price imply? What could cause adoption to fall significantly short?
```

```text
Use cathie-skill to compare [A], [B], and [C].
Rank by S-curve position, cost curve advantage, 5-year TAM exposure, capital requirements, and main risk.
Explain why the top-ranked company earns that position.
```

## Research partner / learning mode

```text
用 cathie-skill 陪我分析 [技术/公司]。
不要直接写报告，每轮先给判断，再问我一个最重要的问题，
带我从颠覆逻辑拆到 S 曲线位置、成本曲线和 5年情景估值。
```

```text
Use cathie-skill as a research partner on [technology / company].
Don't write a report. Each turn: give your judgment, then ask me one focused question.
Walk me from the disruption logic through S-curve position, cost curve, and 5-year valuation.
```
