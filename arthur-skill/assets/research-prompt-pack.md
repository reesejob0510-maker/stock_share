# Research Prompt Pack

Pre-built prompts for Arthur.skill research tasks.

---

## Prompt 1: Full USD Liquidity Assessment

```
用 Arthur Hayes 的框架，分析当前的美元流动性环境。
请评估：
1. 美联储资产负债表的现状和近期趋势
2. 财政部 TGA 余额的变化方向
3. 纽约联储隔夜逆回购（RRP）的动态
4. 三个指标的综合净流动性信号
5. 基于上述判断，对 BTC 的宏观配置含义
```

## Prompt 2: Funding Rate Snapshot

```
分析当前 BTC 永续合约的资金费率情况：
- 当前年化资金费率是多少？与历史分位相比如何？
- 持仓量（OI）是在上升还是下降？
- 是新多单主导、轧空行情、还是多单清算？
- 结合当前流动性环境，资金费率传递了什么信号？
```

## Prompt 3: On-Chain Leverage Risk Check

```
评估当前链上杠杆风险：
- AAVE / Compound 的稳定币借贷年化利率是多少？
- BTC 最近的链上清算密集区在哪里？距当前价格有多远？
- 综合杠杆风险评级是什么：低 / 适度 / 高 / 危险？
- 如果价格下跌 X%，会触发什么规模的链式清算？
```

## Prompt 4: Pre-FOMC Positioning Analysis

```
距下次 FOMC [日期] 还有 X 天，帮我分析：
- 当前市场对 Fed 决定的隐含预期是什么（CME FedWatch）？
- 资金费率是否已经压缩（典型的 FOMC 前去杠杆行为）？
- "会议后行情"的条件是否成立（低资金费率 + 流动性正向）？
- 最大的意外场景是什么？
```

## Prompt 5: Debt Ceiling / TGA Drawdown Analysis

```
当前美国债务上限情况如何？
- TGA 现在是在下降（看多）还是在上升（看空）？
- 这是否是债务上限约束导致的 TGA 消耗阶段？
- X-date 的预计时间是？
- 债务上限解决后，TGA 重建对加密市场的短期影响是什么？
```

## Prompt 6: Macro Event Impact Analysis

```
[具体事件：如"美联储意外降息25bp"/"CPI 数据超预期"/"SEC 拒绝比特币 ETF"]
请用 Hayes 的宏观框架评估这个事件对加密市场的影响：
- 对 USD 流动性的直接影响是什么？
- 对资金费率和链上杠杆的可能连锁反应是什么？
- 短期（1–2 周）和中期（1–3 个月）的影响有何不同？
- 这个事件是否改变了整体风险姿态？
```

## Prompt 7: Altcoin Cycle Phase Check

```
当前处于哪个加密周期阶段？
- BTC 资金费率 vs 主流山寨币（ETH、SOL、BNB）资金费率的分化情况

- 这是早期扩张（BTC 主导）、中期（ETH 跟上）还是晚期（山寨爆发 + 杠杆极值）？
```
