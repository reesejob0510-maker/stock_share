# Research Prompt Pack

Pre-built prompts for Raoul.skill research tasks.

---

## Prompt 1: Full GLI Assessment

```
用 Raoul Pal 的框架，评估当前全球流动性指数（GLI）的状态。
请分析：
1. 美联储、欧央行、日本央行、中国人民银行的当前货币政策方向
2. 全球 M2（美元计价）的同比趋势
3. 信贷脉冲的方向（正向/负向/回升）
4. 综合 GLI 阶段判断：扩张/顶部/收缩/底部
5. 当前 GLI 阶段对跨资产配置的含义

⚠️ 请注明哪些结论是基于公开数据推演，哪些需要 GMI 数据才能确认。
```

```

- Raoul 的"货币贬值必然性"论断在当前环境是否仍然成立？
```
