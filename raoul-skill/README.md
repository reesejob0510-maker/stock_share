## ⚠️ 重要说明：方法论 vs 完整 Skill

Raoul Pal 的核心研究发布于付费机构服务 **GMI（Global Macro Investor）**，包含：
- 专有全球流动性指数（GLI）模型与图表

**本 Skill 不复制 GMI 内容。** 它仅基于 Raoul Pal 在 Real Vision 采访、X（推特）和公开演讲中表达的**公开方法论**。输出结论标注为"方法论推演"，而非 GMI 机构研报。

这使得 raoul-skill 在五位投资人 Skill 中属于**方法论类型**（methodology），而非完整的数据驱动 Skill。

---

## 这个 Skill 做什么

Raoul.skill 将你的 AI Agent 变成一个**全球宏观流动性周期**解读伙伴。

核心逻辑：**全球流动性指数（GLI）是所有市场的母亲**。当主要央行同步扩张时，一切上涨；当它们同步收紧时，一切下跌。比特币是对这一流动性浪潮最敏感的工具，因为它是全球可达、固定供应、无对手方风险的流动性 Beta 资产。

---

## 分析路径

```
全球主要央行货币政策方向（Fed + ECB + BOJ + PBOC）
  → 全球 M2（美元计价）趋势
  → 信贷脉冲（新增信贷占 GDP 比例变化速度）
  → GLI 周期阶段（扩张/顶部/收缩/底部）
  → 人口-债务长周期背景（央行是否被迫最终宽松）
  → 当前叙事窗口识别（宏观 Beta / 技术叙事 / 监管叙事）
  → 跨资产配置含义
  → 什么情况说明 GLI 阶段逆转
```

---

## 四个 GLI 阶段

| 阶段 | 特征 | BTC 含义 |
|---|---|---|
| **扩张** | 多央行同步放水，M2 加速，信贷脉冲为正 | 强力顺风；早期积累窗口 |
| **顶部** | 扩张放缓，央行收紧信号，信贷脉冲回落 | 晚周期；开始减杠杆 |
| **收缩** | 主动紧缩，M2 下滑，信贷脉冲为负 | 显著逆风；70-80% 历史回撤 |
| **底部** | 紧缩结束，领先指标回升，信贷脉冲触底 | 最佳长期买入窗口；BTC 提前 6-12 月领涨 |

---

## 长周期背景：人口 + 债务 = 货币贬值必然性

Raoul 的结构性论断：

1. **婴儿潮一代退休浪潮** → 工作年龄人口占比下降 → 私人消费需求结构性走弱 → 政府被迫赤字扩张
2. **全球债务/GDP 历史高位** → 利率上升空间受限（债务偿还压力）→ 央行无法持续紧缩
3. **每次危机需要更大规模货币响应** → 法币购买力长期趋势性贬值
4. **BTC 固定供应 + 全球可达** → 货币贬值最纯粹的对冲工具

这不是战术判断，而是数十年维度的结构性趋势。即便 GLI 短期收缩，这一结构性偏向仍然成立。

```
GLI 阶段：温和扩张
- 美联储：QT 放缓（信号：正向）
- ECB：维持高利率（信号：中性）
- BOJ：YCC 微调风险（信号：潜在负向，套息平仓风险）
- PBOC：降准降息（信号：正向）
→ 4 家央行中 2 家正向，1 家中性，1 家风险 → 温和扩张

全球 M2（美元计价）：同比 +4.2%，趋势回升
信贷脉冲：中国为正上升，美国为负但触底信号初现

长周期背景：美国债务/GDP 124%，利率敏感度极高
→ Fed 持续紧缩概率低；结构性宽松偏向完整

叙事窗口：宏观 Beta 主导（BTC 与 SPX 相关性 0.72）

跨资产含义：
- BTC：战术超配（GLI 扩张早期最佳 Beta）
- 黄金：超配（实际利率趋于负向）
- 美元：低配（GLI 扩张 + 美元走弱预期）
- 长期国债：中性（等待信用脉冲确认）
```
